Documentation

1. Installation

Run the sovereign installer on any Linux, Mac, or Windows machine:

curl -sSL https://askphane.com/install.sh | bash

2. Connecting Notion

To link your Notion workspace, use the built-in skill command:

phane skill add notion

Note: Your API key is encrypted with AES-256 GCM before being saved to the local vault.

3. Local AI (Ollama)

PHANE automatically detects local Ollama instances on port 11434.
